What TK SteelBot can do

Grade Intelligence

Instant answers on EN series, SAE grades, chemical composition, mechanical properties, equivalents, and application suitability.

Structured RFQ Capture

Guided conversational flow that collects product, grade, size, quantity, application, timeline, and special requirements — all without friction.

Buyer Qualification

Automatically captures contact details and key qualification data (application, urgency, volume) so your sales team receives pre-qualified leads.

WhatsApp Handoff

One-click handoff to your existing WhatsApp sales number with the full RFQ reference pre-filled.

24×7 Availability

Responds instantly to high-intent buyers outside business hours — when many export and forging customers are active.

Lead Capture & Summary

Generates clean, copyable RFQ summaries with reference numbers. Perfect for CRM entry or direct forwarding to the sales team.

Deploy this demo yourself

This is a 100% static single-file website. Perfect for GitHub Pages, Vercel, Netlify, or any static host.

Deploy to GitHub Pages in under 3 minutes:
  1. Download or clone this repository
  2. Copy the tk-steels-ai-assistant/ folder to your repo
  3. Go to your repo → Settings → Pages
  4. Set Source to Deploy from a branch → select the folder tk-steels-ai-assistant
  5. Save and wait ~1 minute. Your demo is live at yourusername.github.io/repo-name
No build step required. No backend. No API keys. Works offline after first load.
